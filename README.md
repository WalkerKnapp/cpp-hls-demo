# cpp-hls-demo
A small demo of a C++ program to stream video data to web clients, using Drogon and FFMpeg

## Requirements 

```shell
sudo apt install git gcc g++ cmake \
                 libboost-all-dev \
                 ffmpeg \
                 libjsoncpp-dev \
                 uuid-dev \
                 openssl libssl-dev \
                 zlib1g-dev
```
